# 블록의 수량 산출
Back:
줄눈 10mm일 때, 1m2당 정미량 12.5매. 소요량은 13매

우선 블록 한 개를 쌓을 때 면적(줄눈 10mm 포함)=(0.39+0.01)×(0.19+0.01) = 0.08(m²)이므로  
단위면적(m²)을 쌓는데 필요한 블록의 정미량 = 1/0.08 = **12.5(매)** 이다.

[[PARA/10. P-Project/현장 관리/벽돌\|벽돌]] 과 마찬가지로 높이가 1m가 아니라면 12.5매 x 높이 하면 된다.

소요량은 할증 4%를 곱하여 산출한다. 12.5×1.04 = **13(매)** 이다.

부자재(모르타르, 시멘트, 모래 등)

블록공사도 쌓는 데 필요한 부자재인 모르타르, 시멘트와 모래를 별도로 산출해야 합니다.
블록쌓기용 모르타르 수량은 블록의 규격, 두께별로 구분하여 줄눈나비 10mm를 기준으로 m²당 필요한 모르타르 수량을 산출합니다.
