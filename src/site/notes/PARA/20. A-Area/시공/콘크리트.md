- 콘크리트의 [[피복\|피복]] : 철근과 콘크리트의 관계 때문에 적정 피복 두께를 유지하는 게 중요하다.
- **슬럼프** : 날씨 영향 고려 (날씨 더우면 조금 더 질게)
	- **호스를 길게 연결 시** (241212 호스 2개) **압송 중 막힐 수** 있다.
	- 유도제를 쓸 수도 있지만, **통수용 몰탈**이 더 성능 좋은듯. 슬럼프 150짜리여도 160~170 정도로 높이는게 좋다.
- **물량**은 직접 뽑아본다. 그래도 100% 정확할 수는 없다.
	- 이유: 슬래브 거푸집은 쳐진다, 벽체 거푸집도 배가 부른다(=측압), 
- **배관 타설** 시 골재량을 무조건 줄여야 한다. 안 그러면 자꾸 막혀서 작업이 안 된다.
	- 배관의 경우 6" 가 아니라 **8"** 로 해야한다. (6"는 골재량을 줄여도 계속 막힘.)
	- 배관 무게 때문에 작업성은 6"가 더 좋지만, 8"로 하는 것이 안전하다.
- 코너 부위에는 [[PARA/10. P-Project/현장 관리/멘끼, 콤비, 휠라 스페이서\|멘끼]] 를 써야한다.
- 더울때([[하절기\|하절기]])는 살수, 추울때([[PARA/20. A-Area/시공/동절기 공사/동절기\|동절기]])는 보온 양생을 해야한다

# 무근콘크리트
- 정의 : 철근이 없고 와이어메쉬만 사용하여 타설하는 콘크리트
- 과정 : 레미콘사 통해 주문 → 현장에서 펌프카, **배관** 등 준비 → 타설
- 사용처 : [[누름\|누름]] ([[PARA/00. Inbox/방수공사\|방수]]층 보호, 구배 형성)

## 지하주차장 타설 시의 예
- 타설 구획 확인 : **"최하층"** 만 하면 됨. (e.g. 해운대처럼 B2F, B3F 있는 경우)
- 타설 전 적정 시공 확인 : 
	- **배수판** (테이핑, 깨진 부위 보수)
	- **PC 트렌치** : 설치높이 및 수평 확인, 균열 여부, **타설 전 보양** 확인
	- **PVC 파이프** 묻힘 위치 확인 (무근 타설 후 위치 확인 곤란)

## 주의사항
- **배관** 직경은 6인치 말고 무조건 **8인치 이상** (6인치는 계속 막힐수밖에 없음)
	- 8인치 배관은 대신 작업성은 안좋음 (무겁다)
- 레미콘사에 배관 타설이라고 무조건 말해줘야 함 (**자갈의 양** 조절, **슬럼프** 질게)
	- 슬럼프가 너무 질면 재료분리 되고, 너무 되면 배관이 막힘.
- **대기시간**을 조절해야 함
	- 배관이 막히면 뚫는 시간, 타설 소요시간, 나라시 따라오는 시간 등 고려
