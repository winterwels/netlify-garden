# Abstract
- 왜 글쓰기가 Area 인가?
	- 일정 복잡도 이상의 생각들의 덩어리는 글쓰기를 통해서만 소화할 수 있기 때문이다. - Luhmann 의 글 [[PARA/00. Inbox/메모 상자와 소통하는 방법\|메모 상자와 소통하는 방법]] 
		- 따라서 **평생** 해야하는 것이다
	- 글쓰기를 통해서 명료화한 Idea는 귀중한 기록이다 - 타인에게 공유가 쉬운 것은 물론, 차후 망각했을 시에 나 자신이 이해하는 데에도 사용할 수 있다.
	- 글쓰기를 통해서 더 오래 기억할 수 있고, 내 생각의 폭이 넓어진다.
		- 즉, 진정한 학습은 글쓰기를 통해서만 가능하다. ※ 참고 : [[PARA/00. Inbox/좋은 답은 좋은 질문에서 나온다\|좋은 답은 좋은 질문에서 나온다]]
