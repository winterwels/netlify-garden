명확한 기한과 목표가 있는 것은 아니지만, 나의 주도 하에 지속적으로 관리하여야 하고, 일정한 관리의 기준 (e.g. "건강 관리" 의 경우, 건강검진에서 주의가 3개 이상 나오지 않게 하는 것 등) 이 있는 것들을 Area에서 관리한다.

# Area 영역의 종류
- [[PARA/10. P-Project/공부\|공부]] : 영어, 중국어, 작곡, 피아노 등.
	- [[PARA/00. Inbox/Anki\|Anki]] : 학습의 근간 (최초 학습 → 반복 경험 → 복습 → 체화)이므로 공부의 Sub-Area로 지정한다.
	- [[PARA/20. A-Area/궁전\|궁전]] : [[기억력\|기억력]] 이 매우 나쁘기 때문에 [[PARA/00. Inbox/기억술\|기억술]] 사용은 필수적이다. 장기기억화 (Anki 등 반복학습을 통한) 되기 전까지 보관하기 위함이다.
- [[PARA/20. A-Area/시공/시공\|시공]] : 시공 공부는 꾸준히 해야한다. (일단은 그게 내 직업이 될테니까)
- [[PARA/20. A-Area/글쓰기/글쓰기\|글쓰기]] : 여러가지 창작. 글쓰기를 포함.
- [[PARA/20. A-Area/TRPG/TRPG\|TRPG]] : 글쓰기와 비슷한 결을 가지는 취미.
- [[PARA/20. A-Area/작곡/작곡\|작곡]] : 재미있는 취미. 요즘은 잘 안하긴 함.
- [[PARA/20. A-Area/습관\|습관]] : [[PARA/10. P-Project/자기관리/ADHD/ADHD\|ADHD]] 관리는 평생 해야한다. 그렇다는건 행동치료의 일환인 [[PARA/20. A-Area/습관\|습관]] 관리도 평생 해야한다는 뜻이다.
	- [[PARA/20. A-Area/감사\|감사]] : 감사하는 습관은 매우 중요하다. 감사일기 포함.
- [[PARA/20. A-Area/건강\|건강]] : 정신건강, 신체건강 (운동 포함)
