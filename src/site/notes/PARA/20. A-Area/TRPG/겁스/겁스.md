# 부딪기 (p.371) 36
양자에게 (HP*속도/100)d의 때리기 피해
- 0.25d = 1d-3
- 0.5d = 1d-2
- 0.5<=x <1d = 1d-1
- 1d 이상 = 소수점 반올림
