# 스펙
[[PARA/40. H-Archive/Cataclysm DDA\|CDDA]] 설정 참고.

## 전력과 충전
- 배터리 탑재
- 케이블 충전 : **엉덩이 위**에 복합 포트 설치. 점퍼 케이블로 **차 배터리**에서 충전 가능.
- 에탄올 반응기 : 술을 전기로 교환 가능하나, 술에 취함.
- 관절 물리력 흡수 장치 : 관절에 톱니바퀴가 있음
- 신진대사 전환기 : 소화기관에 전력 공급장치를 연결.

## 패시브
- 반사신경 개선 : 뇌에서 근육까지 전해지는 명령을 훨씬 더 촉진시키는... (요추 신경에 근육 신경 자극기가 설치됨)
- 향상된 청력 : 인공 와우같이 생긴 바이오닉에 의해 청력이 10배 증가. 과민한 큰 소리도 완충해줌. ![2330301_7399_736.jpg](/img/user/Attachments/2330301_7399_736.jpg) 
- 내부 저장소 : 가슴속에 빈 공간을 만들었음. 부피 2L.
- 냬장형 도구모음 : **왼팔**에 탑재. (망치, 드라이버, 렌치, 용접기, 스패너, 메스 등.)
- 조준 시스템 : 눈에 거리 측정기 설치됨. 사용자가 인지 못해도 자동으로 조준됨.
- 눈 보호 렌즈 : 눈이 외부로부터 완벽히 보호됨. 눈의 눈물길이 입으로 재연결되어 눈물 대신 침이 흐름.
- 감각 감쇄 : 몸에 신경계가 뇌에 전달되는 통증 신호를 억제할 수 있도록 변형. **전력 소비하여 고통을 줄일 수** 있음. 

## 액티브
- 시간 팽창 : 몸의 반응 속도, 이동 속도가 극한. 팔, 다리, 몸통에 1/3 확률로 피해 + 1/5 확률로 텔레포트 멀미
- 전자석 장치 : 손바닥에 전자석 내장. (쇠로 된 물건 당겨올 수 있음)
- 충격파 생성기 : 적을 4m 만큼 밀어냄. 소음 권총 수준 (64)
- 유압 근력장치 : 힘 20 증가됨. (호랑이 수준?)
