# Abstract
짧은 피아노 멜로디를 기반으로 만들게 되었다
전형적인 '산으로 간' 프로젝트였다

# 왜 산으로 갔는가?
1. 메인 멜로디는 피아노다
2. 근데 Pad와 Pluck 신스가 더 도드라져보인다
3. 중간에 느려지는 부분이 너무 길고 지루하다 ("긴장"과 "고조" 되는 부분임에도)
4. 총 소리가 나오면서 긴장을 "해소" 했지만, 해소 치고는 너무 김 빠진다

여러모로 피드백을 많이 받을 법 하다.
