# 프로세스
작곡은 다음 순서대로 이루어진다

- [[작곡/프로듀싱\|프로듀싱]] : 곡의 기반, 즉 멜로디나 전체적 느낌을 만든다
- [[작곡/믹싱\|믹싱]] : 각 악기들의 소리와 기타 상세를 조정한다

※ VST, 샘플 등은 [[작곡/프로듀싱\|프로듀싱]] 에서 다룬다.
※ 모든 앱에 앞서 공통되는 사항들 (e.g. Gain, Attack, Delay 등은 [[작곡/믹싱\|믹싱]] 에서 다룬다.)

## [[PARA/10. P-Project/linux\|linux]] 기본 세팅
[How to install and use VST Plugins on Linux](https://linuxmusician.com/how-to-install-and-use-vst-plugins-on-linux/) 
VST plugins come in two versions:
- **VST2**: Older format, widely supported.
- **VST3**: Newer format with improved performance and features.
Linux supports both formats, but VST2 has broader compatibility with Linux-based DAWs (Digital Audio Workstations).

- **Update your system:** `sudo apt update && sudo apt upgrade`
- **Install essential audio libraries:** `sudo apt install wine-stable winetricks jackd qjackctl`
- **Install your preferred DAW:** Popular options include Ardour, Bitwig, Reaper, and LMMS.

Native VST plugins are designed to work directly with Linux systems.
1. **Download the plugin:** Visit the developer’s site and download the Linux version.
2. **Extract files:** If the file is compressed (.tar.gz, .zip), extract it: `tar -xvzf plugin-name.tar.gz`
3. **Move the plugin to the VST folder:** `sudo mv plugin.so /usr/lib/vst/`
4. **Refresh your DAW’s plugin list:** Open your DAW and scan for new plugins.
