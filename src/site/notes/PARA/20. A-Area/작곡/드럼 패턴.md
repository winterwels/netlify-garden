# 드럼의 종류
보통 드럼이라고 하면 이것들은 무조건 있다.
- 킥 (Kick) : 페달을 밟아서 연주하는 베이스 드럼 [나무위키 - 베이스드럼](https://namu.wiki/w/%EB%B2%A0%EC%9D%B4%EC%8A%A4%EB%93%9C%EB%9F%BC) 
- 스네어 (Snare) : 짧고 높은 소리를 내는 작은 북 [나무위키 - 스네어](https://namu.wiki/w/%EC%8A%A4%EB%84%A4%EC%96%B4%EB%93%9C%EB%9F%BC) 
- 하이햇 (Hi-Hats) : 두 개의 심벌을 걸어놓고 페달로 붙는 정도를 조정할 수 있는 세트. [나무위키 - 하이햇](https://namu.wiki/w/%ED%95%98%EC%9D%B4%ED%96%87) 
	- 페달을 밟지 않으면 두 심벌이 서로 붙지 않는다. 이 상태에서 치면 오픈 햇(Open hat) 이다.
	- 페달을 밟으면 두 심벌이 서로 붙는다. 이 상태에서 치면 클로즈드 햇(Closed hat) 이다.
	- 드럼 샘플에 따라 반쯤 밟는 Half-Closed Hat도 있더라.

이 세 가지 요소는 가장 기본이 되는 요소로서, 주요한 리듬감을 좌우한다.

## 세부적 종류
위에 더해서 이런 것들이 있다.
- 탐: small, medium, large, floor tom 이 있음. [나무위키 - 톰톰](https://namu.wiki/w/%ED%86%B0%ED%86%B0) 
- 카우벨 : [나무위키 - 카우벨](https://namu.wiki/w/%EC%B9%B4%EC%9A%B0%EB%B2%A8) 제일 잘 알려져 있는건 808의 카우벨 샘플. [Youtube](https://www.youtube.com/watch?v=yJLEEjETOME) 
- 스틱 : 드럼 스틱끼리 부딪혀서 나는 꽉 찬 소리. 
- 탬버린 : 우리가 알고있는 그 탬버린. [나무위키 - 탬버린](https://namu.wiki/w/%ED%83%AC%EB%B2%84%EB%A6%B0) 
- 셰이커 : 루프로도 많이 사용하지만 one shot으로도 쓴다. [나무위키 - 셰이커](https://namu.wiki/w/%EC%85%B0%EC%9D%B4%EC%BB%A4#%EC%95%85%EA%B8%B0) 
- 콩가 : 퍼커션으로 간간히 쓴다. [나무위키 - 콩가](https://namu.wiki/w/%EC%BD%A9%EA%B0%80#%EC%95%85%EA%B8%B0) 
- 심벌즈 : 정확히 말해서 하이햇이 심벌즈의 한 종류이다.
	- 크래쉬 (Crash) : 강력한 쏘는 음과 작은 여음을 냄.
	- 라이드 (Ride) : 재즈에서 많이 사용.
