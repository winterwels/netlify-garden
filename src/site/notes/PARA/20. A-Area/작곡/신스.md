오잉? 아까전엔 완만한 사인 곡선 모양이었는데 이번에는 비교적 각진 직선 모양이다. 어떻게 차이가 나는걸까?

![](https://i.imgur.com/TkORNW3.png)
- 킥은 거의 사인파로 이루어져 있고, 처음부터 일관되게 높았다가 낮아지며, 소리의 밀도가 높다. 즉, 귀에 일관되게 들리는 어떤 **음색(Timbre)** 가 있다는거다. 그리고 **낮은 주파수 (Low Frequency)** 를 가지고 있기에 소리도 저음이다.
- 벨은 높낮이도 불규칙하고 어떤 일정한 모양도 안 보인다. 이는 아주 복잡한 **배음구조** (=Inharmonic)를 가졌고, **음색**이 일관적이지 않으며, 날카롭다는 뜻이다. 스플라인이 낮은 각진 곡선 모양인데, **높은 주파수**를 가지고 있기 때문에 그렇다. 그래서 고음이다.

자연에서 나는 여러 소리들은 대부분 이런 식으로 생겼다. 그런데 사인 곡선 모양만 있는 건 아니다.
예를들면, **Sawtooth Wave (톱니파)** 가 있다. 바이올린과 같이 현을 마찰하여 소리를 내는 악기는 이런 톱니파의 소리를 낸다.
**Square Wave (사각파)** 나 **Pulse Wave (펄스파)**, **Triangle Wave (삼각파)** 등도 관악기 따위에서 관찰할 수 있다.

이처럼 다양한 소리들을 모방하기 위해서 탄생한 것이 Synthesizer 인 것이다.

여기까지 이론적 설명이었지만 (교장 선생님 1절만요) 어떤 파형이면 어떤 소리가 난다는 원리를 알아두면 신스 합성하는데 요긴하다.

# 신스의 기본 개념
신스는 여러가지 종류가 있다만, 공통적으로 이런 것들이 있다.
- Oscillator : 기본 파형의 소리를 계속해서 내는 애.
- Filter : 특정 소리만 통과시키는 애. LP(로우패스), HP(하이패스) 등.
