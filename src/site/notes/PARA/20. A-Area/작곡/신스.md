# 소리의 기본 개념
[[소리에 대해서\|소리에 대해서]] 참고. 교장 선생님 훈화같은 글이지만 대충 훑어보면 신스의 특성에 대해 파악하는 데 큰 도움이 된다.

그리고 소리공학도 학문이다. 기본 용어 정리를 안하면 혼동의 여지가 다분하다.

# 신스의 기본 개념
소프트웨어 신스는 여러가지 종류가 있다만, 대개 공통적으로 이런 개념들이 있다.
- Oscillator : 기본 파형의 소리를 계속해서 내는 애.
- Filter : 특정 소리만 통과시키는 애. LP(로우패스), HP(하이패스) 등.
- LFO (Low-Frequency Oscillator) : 일정 주기를 가지고 반복하는 LFO를 이용해 다른 Parameter를 조절한다. (e.g. 필터) 소리를 내는 Oscillator와 달리 비교적 낮은 주파수를 가짐 (=적은 빈도로 진동).
- [[PARA/20. A-Area/작곡/ADSR Envelope\|ADSR Envelope]] : 파형과 더불어 소리가 어떤 특성을 가지는지를 결정함.
