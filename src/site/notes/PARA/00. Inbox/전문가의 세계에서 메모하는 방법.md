# 인용

  ▪ 메모한 내용이 언제, 어떻게 사용될지 아무도 알려주지 않는다.
  ▪ ‘테스트’는 어느 때든, 어떤 형태로든 실시될 수 있다.
  ▪ 메모를 잘 했다면 언제든지 참조할 수 있다.
