# 배합설계표
슈퍼프린트라고도 함.

## 종류
1. 일일 배합표 : 둘 다 보관
	1. 타설 직후 주는 것 : 최초 발급. 맨 마지막과 대비해서 변동 가능성 있는듯?
	2. 타설 후 며칠 뒤에 주는 것 : 최초 발급본 대비 페이지 수가 많음. (=좀더 상세)
2. 설계배합표 : 공장점검 및 배합설계 할 때 하는 것. 배합시험결과보고서에 포함되어 있음.
