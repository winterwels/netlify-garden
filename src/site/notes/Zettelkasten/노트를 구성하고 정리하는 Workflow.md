# 노트를 구성하고 정리하는 Workflow
- [노트를 구성하고 정리하는 Workflow](https://cafe.naver.com/obsidianary/13714)
	- 작성자만의 방법임 (MOC 참고: `Nick Milo의 LYT(Linking your thinking)`)
	- 1. 노트 작성: INBOX → Dataview (노트 생성 후 경과일 체크) (**작성**)
	- 2. 폴더 안에 노트등 옮기고, 동일 폴더 안에 Guide Note 생성 (**분류**)
	- 3. 연관성 있는 노트를 하나의 Guide Note로 연결 (**연결-1형**)
		- Properties에 `up` 필드 생성 → 필드에 `[[link]]` 입력
			- Nick milo 동영상 참고
		- 또 다른 디렉토리에서 참고할 자료 검색 (**연결-2형**) 
	- 4. Dataview와 Link로 군도`archipelago` 형성 (**시각화**)
	- 5. 관련된 노트 요약 → 하나의 완결된 이야기 형성 (**요약**)
	- Home Note : 여러 MOC에 접근하기 용이하게끔 만든 노트
		- 경로: PARA 분류체계 안에 각 Guide 노트 넣어놓음. → 각 PARA 폴더 안에 Guide 노트를 모아볼 수 있게 Home note 형성
