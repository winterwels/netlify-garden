# 정체성
정체성이란 내가 어떤 사람이라는 의식을 말한다. 성정체성, 소수자적인 성지향성, 정신병을 인식하고 스스로의 정체성으로 삼는 것 (예를들어 신경다양인) 등이 있겠다.

보통 사람들은 정체성이라는 것에 대해 경시하지만, 사실 자신이 어떤 정체성을 가졌는지 아는 것은 중요하다. 예를들어 장애가 그렇다. [[PARA/00. Inbox/장애는 생각보다 보편적이다\|장애는 생각보다 보편적이다]] 에서 보다 자세히 기술한다.
