# 설명하지 말고 상상하게 하라
[[PARA/20. A-Area/TRPG/TRPG\|TRPG]] 의 대원칙 중 하나이다. 배경에 대해서 설명하지 말고 그 배경이 인물들에게 어떤 느낌으로 다가오는지, 어떤 흥미로운 것들이 있고 어떤 이야기가 펼쳐지는지. 흥미로운 것부터 순서대로 묘사해야한다.

아니, 사실 모든 창작의 원칙이다. 대부분의 독자들은 작가가 대놓고 뭔가를 가르치려 들면 좋아하지 않는다. (좋아하는 사람도 있긴 하다)

[[설득\|설득]]에 대해서도 마찬가지이다. 가령 당신이 내 제안을 따른다면 어떤 이득이 있을 것이라고 한다면, 단순히 설명하지 말고 그 이득이 얼마나 달콤할지, 얼마나 쉬운 선택일지에 대해 묘사하는게 더 낫다. -feat. 지미 맥길
